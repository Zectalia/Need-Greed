📜 NEED & GREED - RÈGLES OFFICIELLES

🎮 LE JEU EN BREF

C'est un jeu de descente de donjon dans lequel les joueurs vont devoir coopérer afin d'obtenir du butin. Ce même butin va leur offrir de nouveaux effets lors de leur descente en donjon, et c'est aussi ce même butin qui permettra aux joueurs d'obtenir la victoire. Régulièrement, les joueurs auront la possibilité de vendre leurs butins pour augmenter leur richesse, le joueur le plus riche à la fin de la partie est déclaré vainqueur.

⚙️ PRÉPARATION DU JEU

  1. Placer le plateau "Donjon"
  2. Chaque joueur sélectionne une carte héros, un dé et des jetons de vote. Ils définissent leur roue de PV sur "15".
  3. Les joueurs installent ensuite leur carte autour du plateau de jeu "Donjon"
  4. Chaque joueur lance son dé, le joueur qui fait le plus grand score décide de l'ordre des joueurs en leur distribuant les jetons de rôle suivants : Tank / CàC / Dps Distance / Soutien
  5. Retirez les monstres à 3 coffres. Sélectionner en deux aléatoires à placer lors de la descente 3
  6. Pour commencer votre descente :
    - Descente 1 : 8 monstres + 12 événements
    - Descente 2 : 12 monstres + 16 événements
    - Descente 3 : Mélanger descentes 1 et 2 + monstres à 3 coffres

⚔️ DÉROULÉ DE LA PARTIE

Lorsque tout est en place, le joueur qui a obtenu le plus haut lancer de dé pioche une carte dans la pioche et la retourne face verso sur le plateau donjon.

  • Cartes non-monstres : Résolvez l'effet et piochez la carte suivante.
  • Carte Monstre : Entrez en phase de combat.

⚔️ PHASE COMBAT

Ordre des jetons : Tank → CàC → DPS Distance → Soutien.
Monstres à 1 coffre agissent en premier, monstres à 2 coffres après 2 actions joueurs, monstres à 3 coffres après 4 actions joueurs.

  • Tour joueur : lance un dé → dégâts infligés au monstre
  • Tour monstre : lance le nombre de dé indiqué + bonus aux dégâts
  • Joueur à 0 PV : conserve un butin < 2 lingots d'or, perd le reste + moitié richesse, devient spectre
  • Monstre à 0 PV : le joueur qui a infligé le coup fatal pioche le nombre de butins indiqué → phase Looting

💰 PHASE LOOTING

Le joueur ayant porté le coup fatal pioche les butins. Il peut les conserver, les donner ou les laisser aux autres.
Ninja Loot : Un joueur peut tenter de voler. Le propriétaire et les voleurs lancent un d6. Le plus haut résultat garde UN butin. Si le Ninja Looter perd, il perd 3 PV.

📦 INVENTAIRE

Chaque héros possède 4 places. Les symboles sur les cases activent les effets des butins. Les trésors n'ont pas besoin de case précise.

🃏 LA PIOCHE

Gérée par le joueur ayant fait le dé le plus haut, puis transférée au vainqueur du combat.
Cartes Événements : phase de vote (jetons vert/rouge). Certaines cartes exigent un nombre précis de votes VERT pour être appliquées. Sinon, piocher la carte suivante.

🏃 LA FUITE / SORTIE DU DONJON

Un joueur peut quitter le donjon après un événement ou un combat. Son inventaire est sauvegardé, il ne participe plus. Quand tous les joueurs sont partis ou morts, la descente prend fin → phase de vente.

👻 LE RÔLE DE SPECTRE

En cas de mort, vous devenez spectre. Vous pouvez : soigner 1 PV (joueur ou monstre), infliger 1 PV, ou mélanger l'inventaire d'un joueur (1 fois par joueur par descente).

🏆 FIN DE PARTIE

Après 3 descentes et les ventes, le joueur le plus riche est déclaré vainqueur !

🎖️ INFORMATIONS SUR LES JETONS RÔLES

  • Tank : Les équipements "Armure" confèrent 1 PV supplémentaire
  • CàC : Les équipements "Arme" confèrent 1 PV supplémentaire
  • DPS Distance : Relance une fois le dé d'attaque si coup fatal possible
  • Soutien : Peut soigner 3 PV au lieu d'attaquer, gagne 1 richesse par soin

Le Leader réorganise la distribution des rôles. En cas de mort/fuite, le jeton est retiré.

🌊 RÈGLE DE CASCADE

Si un monstre cible un jeton absent, la cible suivante dans l'ordre est choisie :
Tank → CàC → DPS Distance → Soutien → Tank
Si un joueur subit plusieurs attaques à cause de cette règle, n'appliquez que la première.

🔄 PRÉCISION ENTRE LES DESCENTES

Les joueurs récupèrent automatiquement leurs PV de base (15) et réappliquent les effets des équipements affectant les PV.

⚠️ EFFETS NÉGATIFS ET BUTIN MAUDIT

Ceux-ci s'appliquent même si le butin n'est pas sur la case appropriée.

📦 GESTION DE L'INVENTAIRE

Vous êtes libre de le gérer, mais il est interdit d'y toucher pendant les combats ou la résolution d'une carte événement.

🤝 NÉGOCIER ENTRE LES JOUEURS

Les joueurs peuvent échanger butins ou richesses librement, sauf pendant les événements.

🔥 PRÉCISION SUR LES BUTINS ENFLAMMÉS

Le joueur peut décider de ne pas activer l'effet d'un butin enflammé.
